ТЕМЫ ЛЕКЦИИ (используй как структуру резюме):
{context}

ТЕКУЩЕЕ РЕЗЮМЕ (дополни его информацией из нового фрагмента):
<current_summary>
{resume}
</current_summary>

НОВЫЙ ФРАГМЕНТ ЛЕКЦИИ:
<new_fragment>
{text}
</new_fragment>

ИНСТРУКЦИИ:
1. Извлеки ключевые идеи, аргументы и примеры из нового фрагмента.
2. Дополни текущее резюме новой информацией, сохраняя структуру по темам.
3. Объедини похожие идеи, избегай повторов.
4. Отмечай важные цитаты и конкретные данные (цифры, названия, имена).
5. Если есть вопросы аудитории по теме — включи ключевые.

ФОРМАТ РЕЗУЛЬТАТА:
- Используй Markdown
- Структурируй по темам из списка выше
- Каждая тема: заголовок + 3-5 ключевых тезисов
- Общий объём: 500-1000 слов
