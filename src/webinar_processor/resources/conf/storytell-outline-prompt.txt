Проанализируй транскрипт и составь:
1. План учебной статьи (4-7 смысловых разделов)
2. Словарь ключевых терминов с каноническим написанием

ПРАВИЛА ДЛЯ ПЛАНА:
1. Разделы должны отражать ЛОГИЧЕСКУЮ структуру лекции (не хронологическую)
2. Если одна тема обсуждается в нескольких местах лекции — объедини в один раздел
3. Организационные части (приветствия, перерывы, техсбои) — НЕ делай отдельным разделом
4. Каждый раздел = одна крупная тема
5. Описание раздела (covers) — 2-3 предложения о том, что именно из транскрипта входит в этот раздел

ПРАВИЛА ДЛЯ ТЕРМИНОВ:
1. Выпиши 10-20 ключевых терминов и концепций из лекции
2. Для каждого укажи каноническое написание на русском и английский эквивалент
3. Если лектор использует разные варианты написания — укажи каноническую форму
