Напиши краткое структурированное резюме лекции (500-1000 слов) в формате Markdown.

СТРУКТУРА:
- # Заголовок (5-9 слов)
- 1-2 предложения: о чём лекция и для кого
- 4-7 тематических блоков, каждый: ## подзаголовок + 3-5 ключевых тезисов
- ## Заключение: 2-3 предложения, главные выводы

ПРАВИЛА:
1. Структурируй по ТЕМАМ, не хронологически
2. Сохраняй конкретику: названия, цифры, примеры
3. Объединяй повторяющиеся объяснения
4. Вопросы аудитории — включай ключевые, если они добавляют ценность
5. Организационные вставки — пропускай
6. Формат: Markdown — используй # для заголовка, ## для подзаголовков, **жирный** для ключевых терминов, списки где уместно
